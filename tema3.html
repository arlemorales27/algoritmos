Git desde cero: sube tu primer repositorio y colabora sin complicarte

Explicaciones cortas, comandos para practicar

Empezar ahora Colaborar en equipo

Primeros pasos (10 minutos)

Necesitas
Git instalado y una cuenta en GitHub o GitLab.
Paso 1
Verifica que Git funciona
Compruebas que está instalado correctamente.
git --version
Paso 2
Preséntate a Git (una sola vez)
Tu nombre y correo salen en los commits.
git config --global user.name "Tu Nombre"
git config --global user.email "tu@correo.com"
git config --global init.defaultBranch main
Paso 3
Crea un proyecto nuevo
Empiezas con un README y tu primer commit.
mkdir mi-proyecto && cd mi-proyecto
git init
echo "# Mi Proyecto" > README.md
git add .
git commit -m "Primer commit: crea README"
Paso 4
Conecta con GitHub y sube (push)
Tu código vive en la nube y puedes compartirlo.
# En GitHub/GitLab crea un repo vacío llamado mi-proyecto
git branch -M main
git remote add origin https://github.com/usuario/mi-proyecto.git
# o por SSH: git@github.com:usuario/mi-proyecto.git
git push -u origin main
Paso 5
Clona un repositorio existente
Descargas una copia local lista para editar.
git clone https://github.com/organizacion/app.git
cd app

Tu día a día (editar → guardar → subir)

Guarda cambios

git status
git add .
git commit -m "mensaje corto y claro"

Sincroniza con la nube

git pull
git push

Colaborar en equipo (lo esencial)

Trabaja en una rama aparte

git switch -c feature/contacto
# ... edita archivos ...
git add . && git commit -m "contacto: agrega formulario"
git push -u origin feature/contacto

Abre un Pull/Merge Request

En GitHub/GitLab crea el PR/MR de tu rama hacia main y pide revisión.

Mantén tu rama al día

git pull origin main
git add .
git commit -m "resuelve conflictos simples"
git push

Errores comunes

  • Olvidar git add . antes del commit.
  • Hacer cambios en main directamente.
  • Mensajes de commit vagos.

Recursos básicos

Ver historial

git log --oneline -n 20

Ignorar archivos

node_modules/
.env
.DS_Store

Ayuda integrada

git help commit
git help push
