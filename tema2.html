Guía de JavaScript 2025

Conceptos fundamentales de JavaScript para desarrolladores modernos

1. Cómo ejecutar código JavaScript

Embebido en HTML

Incluye JavaScript dentro de una etiqueta <script> en un archivo HTML.

<script>
    console.log("Hola, mundo!");
</script>

Archivo externo

Enlaza un archivo .js externo usando src en la etiqueta <script>.

<script src="script.js"></script>

Contenido de script.js:

console.log("Hola desde un archivo externo!");

Con Node.js

Ejecuta JavaScript en el servidor o localmente con Node.js.

node script.js

2. Qué es una variable

Una variable es un contenedor para almacenar datos. Se declara con let, var o const.

let nombre = "Juan";
var edad = 25;

3. Qué es una constante

Una constante es una variable cuyo valor no puede reasignarse tras su declaración.

const PI = 3.14159;
PI = 3.14; // Error: no se puede reasignar

4. Características de los identificadores

  • Deben comenzar con una letra, _ o $.
  • Pueden contener letras, números, _ o $.
  • No pueden ser palabras reservadas (ej. if, for).
  • Son sensibles a mayúsculas/minúsculas.
let $nombre = "Ana";
let _edad = 30;
let nombreCompleto = "Ana García";

5. Tipos de datos en JavaScript

JavaScript es un lenguaje dinámicamente tipado. Tipos principales:

  • number: Números (enteros, decimales, BigInt).
  • string: Cadenas de texto.
  • boolean: true o false.
  • object: Objetos, arrays, null.
  • undefined: Variable no asignada.
  • symbol: Identificador único.
  • bigint: Números enteros de precisión arbitraria.
let numero = 42;
let texto = "Hola";
let esVerdadero = true;
let objeto = { nombre: "Juan" };
let indefinido;
let simbolo = Symbol("id");
let grande = 123n;

7. Trabajo con cadenas de texto

Las cadenas son secuencias de caracteres. Métodos comunes:

  • length: Longitud de la cadena.
  • toUpperCase: Convierte a mayúsculas.
  • toLowerCase: Convierte a minúsculas.
  • slice: Extrae una porción.
let texto = "Hola, Mundo!";
console.log(texto.length); // 12
console.log(texto.toUpperCase()); // HOLA, MUNDO!
console.log(texto.slice(0, 4)); // Hola

10. Alert

La función alert muestra un mensaje en una ventana emergente.

alert("¡Bienvenido a la página!");

11. Conversión de tipos de datos

JavaScript realiza conversiones implícitas o explícitas.

let numero = "42";
let convertido = Number(numero); // 42
let texto = String(42); // "42"
let booleano = Boolean(1); // true

12. Ingresar datos por teclado

Usando prompt

La función prompt permite al usuario ingresar datos.

let nombre = prompt("Ingresa tu nombre:");
console.log("Hola, " + nombre);

Usando un input HTML

Usa un campo <input> para capturar datos.

<input type="text" id="entrada" placeholder="Ingresa algo">
<button onclick="mostrarEntrada()">Enviar</button>
<script>
function mostrarEntrada() {
    let valor = document.getElementById("entrada").value;
    console.log(valor);
}
</script>

13. Qué es depuración

La depuración es el proceso de identificar y corregir errores (bugs) en el código. Herramientas:

  • console.log: Muestra valores en la consola.
  • Debugger: Pausa la ejecución en el navegador.
  • Extensiones como DevTools en Chrome/Firefox.
let x = 10;
console.log("Valor de x:", x);
debugger; // Pausa aquí en DevTools

Tarea: Tipos de datos y operador ternario

Instrucciones: Crea un programa que solicite al usuario su edad (usando prompt) y determine si es mayor de edad usando un operador ternario. Almacena el resultado en una variable y muestra un mensaje con alert. Usa diferentes tipos de datos.

let edad = prompt("Ingresa tu edad:");
let esMayor = (Number(edad) >= 18) ? "Mayor de edad" : "Menor de edad";
alert("Tú eres: " + esMayor);
let info = {
    edadIngresada: edad,
    resultado: esMayor
};
console.log(info);
